Nombre de la función
comparar_svc_lineal_vs_rbf(X, y)
📌 Descripción

Se reciben:

X → matriz numpy

y → vector numpy

La función debe:

Entrenar:

SVC(kernel="linear")

SVC(kernel="rbf")

Calcular accuracy en los mismos datos.

Retornar:

"linear" si el modelo lineal tiene mayor accuracy

"rbf" si el modelo RBF tiene mayor accuracy

"empate" si ambos son iguales
